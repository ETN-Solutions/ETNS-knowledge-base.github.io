### Conditionals

1. If
syntax: 
```rs
let price = 10;
if price > 0 {
    println!("Price is greater than zero");
}
```

2. If else 
syntax: 
```rs
let price = 10;
if(price > 0) {
    println!("Price is greater than zero");
} else {
    println!("Price less than or equal to zero");
}
```
